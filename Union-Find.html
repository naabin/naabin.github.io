Union Find

Union find is as the name suggests it unions (adds) the elements into the subset and find the elements within the subset. It satisfies the following equivalence relation.

  • Reflexive: p is connected to p.
  • Symmetric: If p is connected to q, then q is connected to p.
  • Transitive: If p is connected to q and q is connected to r then p is connected to r.

Union find algorithm is ubiquitous from problems related to Percolation to Connection establishments in Networks and in Social Networks.

API specification (Java)

UF (int n) // initializes n sites with integer names (0 to n - 1)
void union(int p, int q) // adds connection between p and q
int find(int p) // finds the element p connected to root element
boolean connected (int p, int q)
int count() // number of components (subsets)

 

Implementation (Java)

The following implementation is the weighted union find data structure. It means that it will guarantee the depth of the internal nodes will not degenerate(similar to linkedlist) and instead the structure of the tree will be something like forest of trees. two or more nodes may refer to same root node. This will in turn reduce the find() method operation signigicantly.

public class UF {
  private int size;
  private int[] sz; 
  private int[] id;
  
  public UF(int n) {
      if (n < 0) throw new IllegalArgumentException("n should be greater than 0");
      for (int i = 0; i < n; i++) {
        id[i] = i;
      }
      // Initially there are n components and all of them has size of 1.
      for (int i = 0; i < n; i++) {
        sz[i] = 1;
      }
      size = n;
    }
    
    public int find(int p) {
      while (p != id[p]) {
        p = id[p];
      }
      return p;
    }
      
    public void union(int p, int q) {
      int pRoot = find(p);
      int qRoot = find(q);
      if(pRoot == qRoot) return;
      if(sz[pRoot] < sz[qRoot]) {
        id[pRoot] = qRoot;
        sz[qRoot] += sz[pRoot];
      }
      else {
        id[qRoot] = pRoot;
        sz[pRoot] += sz[qRoot];
      }
      size--;
    }
      
    public boolean connected(int p, int q) {
      return find(p) == find(q);
    }
    
    public int count() {
      return size;
    }
}
