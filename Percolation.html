Percolation

From Algorithms-I Princeton University

Percolation refers to movement and filtering of fluids through the porous materials. Percolation theory describes the behavior of network when nodes are added.

Given a porous landscape with water on the surface (or oil below), under what conditions will the water be able to drain through to the bottom (or the oil to gush through the surface)?

Percolation model

In particular we model the percolation system using grid of sites. Each site is either open or blocked. A full site is an open site in the top row via chain of (left, right, top down) open sites. The system is said to be percolated if there is a full site in the bottom row. In other words, a system percolates if we fill all open sites connected to top row and that process fills some open site on the bottom row.

The problem

In a famous scientific problem, researchers are interested in the following question: if sites are independently set to be open with the probability ( and therefore blocked with probability ), what is the probability that the system percolates. When equals to 0, the system does not percolate; when equals 1, the system percolates. When n is sufficiently large, there is a threshold value such that

public class Percolation {
    public[] boolean sites;
    public int numberOfOpenSites = 0;
    //Data structure UF can be found on https://naabin.github.io/#union-find
    public UF uf;
    public UF bw;
    private int top;
    private int bottom;
    privatin int n;
    //creates n-by-n grid, with all sites initially blocked
    public Percolation(int n) {
       if (n <= 0) throw new IllegalArgumentException();
       this.n = n;
       this.sites = new boolean[n * n];
       this.uf = new UF(n * n + 2);
       this.bw = new UF(n * n + 2);
       this.top = 0;
       this.bottom = n * n + 1;
    }
    private int xyTo1d(int row, int col) {
        return this.n * (row - 1) + (col - 1);
    }
    
    //open the site(row, col) if it is not open already
    public void open(int row, int col) {
        if (row <= 0 || row > this.n) throw new IllegalArgumentException();
        if (col <= 0 || col > this.n) throw new IllegalArgumentException();
      int currentIndex = xyTo1d(row, col);
      if (!isOpen(row, col)) this.sites[currentIndex] = true;
      //Connect to top virtual node
      if (row == 1) {
          this.uf.union(this.top, currentIndex);
          this.bw.union(this.top, currentIndex);
      }
      // Connect to bottom virtual node
      if (row == n) {
          this.bw.union(this.bottom, currentIndex);
      }
      if (row > 1 && (row-1, col)) {
        int nextTopIndex = xyTo1d(row-1, col);
        this.uf.union(currentIndex, nextTopIndex);
        this.bw.union(currentIndex, nextTopIndex);
      }
      if (row < n && isOpen(row + 1, col)) {
          int nextBottomIndex = xyTo1d(row + 1, col);
          this.uf.union(currentIndex, nextBottomIndex);
          this.bw.union(currentIndex, nextBottomIndex);
      }
      if (col > 1 && isOpen(row, col - 1)) {
          int nextLeftIndex =xyTo1d(row, col - 1);
          this.uf.union(currentIndex, nextLeftIndex);
          this.bw.union(currentIndex, nextLeftIndex);
      }
      if (col < n && isOpen(row, col + 1)) {
          int nextRightIndex = xyTo1d(row, col + 1);
          this.uf.union(currentIndex, nextRightIndex);
          this.bw.union(currentIndex, nextRightIndex);
      }
      ++this.numberOfOpenSites;
    }
    // is the site(row, col) open
    public boolean isOpen(int row, int col) {
        if (row <= 0 || row > this.n) throw new IllegalArgumentException();
        if (col <= 0 || col > this.n) throw new IllegalArgumentExpcetion();
        int index = xyTo1d(row, col);
        return this.sites[index];
    }
    
    // is the site(row, col) full
    public boolean isFull(int row, int col) {
        if (row <= 0 || row > this.n) throw new IllegalArgumentException();
        if (col <= 0 || col > this.n) throw new IllegalArgumentExpcetion();
        return isOpen(row, col) && this.uf.find(xyTo1d(row, col)) == this.uf.find(this.top);
    }
    //returns the number of open sites
    public int numberOfOpenSites() {
        return this.numberOfOpenSites;
    }
    
    // does the system percolates
    public boolean percolates() {
        return this.bw.find(this.top) == this.bw.find(this.bottom)
    }
}
Monte Carlo Simulation

To estimate the percolation threshold, consider the following computational experiment:

  • Initialize all sites to be blocked.

  • Repeat the following until the system percolates:

    • Choose a site uniformly at random among all blocked sites.
    • Open the site.

  • The fraction of sites that are opened when the system percolates provides an estimate of the percolation threshold.

For example, if the sites are opened in a lattice, then our estimate of the percolation threshold is 204/400 = 0.51 because system percolates when the 204th site is opened.

By repeating this computation experiment times and averaging the results, we obtain a more accurate estimate of the percolation threshold. Let be the fraction of open sites in computation experiment . The sample mean provides an estimate of the percolation threshold; the sample standard deviation ; measures the sharpness of the threshold.

 

Assuming T is sufficiently large (say at least 30), the following provides 95% confidence interval for the percolation threshold.

public class PercolationStats {
    private int trials;
    private double[] count;
    // perform independant trials on an n-by-n grid
    public PercolationStats(int n , int trials) {
        if (n <= 0 || trials <= 0) throw new IllegalArgumentException();
        this.count = new double[trials];
        this.trials = trials;
        for (int i = 0; i < trials; i++) {
            Percolation percolation = new Percolation(n);
            while(!percolation.percolates()) {
                int p = StdRandom.uniform(1, n+1);
                int q = StdRandom.uniform(1, n+1);
                if(!percolation.isOpen(p, q)) percolation.open(p, q);
            }
            StdDraw.enableDoubleBuffering();
            PercolationVisualizer.draw(percolation, n);
            StdDraw.show();
            this.count[i] = percolation.numberOfOpenSites()/(n * n * 1.0);
        }
    }
    // sample mean of percolation threshold
    public double mean() {
        return StdStats.mean(count);
    }
    // sample standard deviation of percoaltion threshold
    public double stddev() {
        return StdStats.stddev(count);
    }
    // low endpoint of 95% confidence interval
    public double confidenceLo() {
        double lo = (1.96 * stddev())/Math.sqrt(trials);
        return mean() - lo;
    }
    // high endpoint of 95% confidence interval
    public double confidenceHi() {
        double hi = 1.96 * stddev()/Math.sqrt(trials);
        return mean() + hi;
    }
    // test client
    public static void main(String[] args) {
        int n = Integer.parseInt(args[0]);
        int trials = Integer.parseInt(args[1]);
        PercolationStats stats = new PercolationStats(n, trials);
        System.out.println("mean                    = " + stats.mean());
        System.out.println("stddev                  = " + stats.stddev());
        System.out.println("95% confidence interval = [" + stats.confidenceLo() + ", " + stats.confidenceHi() + "]");
    }
}
