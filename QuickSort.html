Quick Sort

Quick sort is yet another fundamental fast sorting algorithm invented in early 60's by British computer scientist Tony Hoare . Quick sort follows the same divide and conquer paradigm but has key difference when comparing to merge sort. Quick sort does not take auxiliary space which makes this algorithm in-place as well as it first takes an array element which sometimes referred to as a pivot element and it is chosen in way that elements left to pivot are less and to elements to the right are greater then it divides the array with pivot index and recursively sort the subarrays. Quick sort is not a stable because the long exchanges it makes when it does the partition. Quick sort is fast but not stable means that it may not be suitable for the reference types but again it would entirely depend on the use case.

procedure quicksort(arr, low, high):
    pivot = partition(arr, low, high);
    quicksort(arr, low, pivot - 1);
    quicksort(arr, pivot + 1, high);

procedure partition(arr, low, high):
    i = low, j = high + 1;
    arrayEl = arr[low];
    while true:
        * increment i until elements are less than arrayEl and i is less than high
        * decrement j unitl elements are greater than arrayEl and j is less greater than            low
        * if i is greater than equal j 
            * no swap needed
        * swap(arr, i, j)
    swap(arr, low, j);
    return j;

        int pivot = partition(arr, low, high);
        sort(arr, low, pivot - 1);
        sort(arr, pivot + 1, high);
    }
    
    private static<T extends Comparable<T>> int partition(T[] arr, int low, int high) {
        int i = low, j = high + 1;
        T el = arr[low];
        while (true) {
            while (SortUtil.less(arr[++i], el)) {
                if (i == high) break;
            }
            while (SortUtil.less(el, arr[--j])) {
                if (j == low) break;
            }

            SortUtil.exchange(arr, i, j);
        }
        SortUtil.exchange(arr, low, j);
        return j;
    }
